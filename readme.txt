Corrects http image links on a page delivered over https.

== Description ==

A simple plugin that sets the protocol of a thumbnails source to match the protocol that is currently being used to display a page. This is helpful if you've recently implemented https on your site and your browser is telling you that the page contains items that have been delivered insecurely.

== Frequently Asked Questions ==

= Does this plugin affect non-SSL traffic? =

Yes. If a page is being displayed with http it'll also correct thumbnail urls that start with https and set them to http
